Halo: Combat Evolved

The Game That Defined a Console Generation

Game Information

Release Date: November 15, 2001
Developer: Bungie Studios
Platform: Xbox, Windows, Mac
Metacritic Score: 97/100

Synopsis

Halo: Combat Evolved is a first-person shooter video game developed by Bungie and published by Microsoft Game Studios. It was released as a launch title for the Xbox gaming system on November 15, 2001. The game is set in the twenty-sixth century, with the player assuming the role of the Master Chief, a cybernetically enhanced supersoldier.

Key Features

  • Revolutionary first-person shooter gameplay designed for consoles
  • Epic single-player campaign with immersive storytelling
  • Innovative two-weapon limit and regenerating shield system
  • Groundbreaking multiplayer mode (via System Link)
  • Introduction of iconic characters Master Chief and Cortana
